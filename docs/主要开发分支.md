# 主要开发分支：XLRHopper

本仓库当前以 **`XLRHopper`** 分支作为**主要开发与构建分支**（仅含漏斗插件）。

## 本地开发

```bash
git fetch origin
git checkout XLRHopper
git pull origin XLRHopper
mvn clean package
```

## GitHub 默认分支（建议）

将 GitHub 仓库默认分支改为 `XLRHopper`，便于 PR、Codespace 与克隆时直接进入漏斗项目：

1. 打开仓库 **Settings** → **General**
2. **Default branch** → **Switch default branch**
3. 选择 **`XLRHopper`** → 确认

## 分支说明

| 分支 | 用途 |
|------|------|
| **XLRHopper** | 主要开发：高级漏斗传输插件 |
| **XLRLightweightChat** | 维护：多功能聊天插件（独立历史） |
